## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo

    ```sh
    git clone https://github.com/iaeshliman/jwt.git
    ```

2. Install NPM packages

    ```sh
    npm install
    ```

3. Create a `.env` file to define custom settings

    ```bash
    # Application
    APPLICATION_NAME=
    APPLICATION_PORT=

    # Debug
    DEBUG_DISPLAY_STACKTRACE=
    DEBUG_LOG_LEVEL=

    # Mysql
    MYSQL_HOST=
    MYSQL_USER=
    MYSQL_PASSWORD=
    MYSQL_DATABASE=

    # Jwt
    JWT_SECRET=
    JWT_ISSUER=
    JWT_AUDIENCE=
    ```

4. Start the server

    ```js
    npm run dev
    ```
